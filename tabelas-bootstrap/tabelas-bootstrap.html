Tabelas com efeito hover

The .table-hover class enable a hover state on table row:

Classes Contextuais

Contextual classes can be used to color the table, table rows or table cells. The Classes that can be use are: .table-primary, .table success, .table-info, .table-warning, .table-danger, .table-light, .table-dark, .table-active, .table-secondary
